## 1. 产品概述

智慧机房信息化管理系统官方网站 —— 面向企业客户的产品宣发平台，展示一体化机房运维管理解决方案，通过高质量视觉设计与流畅交互动画提升品牌专业形象，推动商业转化。

- 目标用户：企业IT负责人、机房运维经理、数据中心决策者
- 核心价值：以精致的视觉呈现传达产品的专业性、全面性与可靠性

## 2. 核心功能

### 2.1 用户角色
| 角色 | 访问方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 直接访问 | 浏览全部页面内容、查看产品功能、提交咨询 |
| 管理员 | 后台登录 | 管理内容、查看咨询记录（本次不实现后台） |

4. **关于我们**：公司介绍、团队实力、联系方式
5. **联系咨询**：产品咨询表单、预约演示

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：纯白色背景 (`#FFFFFF`)，搭配深邃黑文字 (`#1D1D1F`)
- **强调色**：科技蓝 (`#0066CC`) 作为主点缀，浅灰蓝 (`#F5F5F7`) 作为区块分隔
- **按钮风格**：圆角胶囊按钮，主按钮填充深蓝，次按钮描边 + 悬停填充
- **字体**：中文使用 PingFang SC / SF Pro Display，英文使用系统衬线增强高级感
- **布局风格**：宽幅留白、大间距、居中对称与不对称穿插结合
- **图标风格**：线性简约图标，悬停时有颜色渐变过渡

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 首页 | Hero 区域 | 超大字号标题、文字渐显动画、背景微妙渐变光晕、底部向下滚动提示 |
| 首页 | 功能概览 | 3×4 卡片网格、卡片悬停轻微上浮 + 阴影加深、图标颜色过渡 |
| 首页 | 数字驾驶舱 | 深色背景区块、动态数据展示、数字滚动计数、折线图动画 |
| 首页 | 客户价值 | 大号数字 + 说明文字、四列布局、交错动画 |
| 功能详情页 | 功能展示 | 左右分栏交替布局、功能图标大尺寸展示、内容渐入动画 |
| 联系咨询页 | 表单 | 输入框聚焦时边框高亮、提交按钮涟漪效果、成功提示动效 |

### 4.3 响应式
- 桌面端优先（1440px 基准）
- 平板端（768px）：布局自适应，卡片网格变为 2 列
- 移动端（375px）：单列布局，汉堡菜单，字号缩放

### 4.4 动效设计指引
- **滚动触发动画**：元素进入视口时渐显上移，使用 Intersection Observer
- **悬停交互**：按钮、卡片、链接均有微妙的颜色 / 位移 / 阴影变化
- **页面过渡**：路由切换时的淡入淡出
- **数字驾驶舱**：数据数字从 0 滚动到目标值，图表线条绘制动画
- **文字效果**：产品名称文字有流光扫过效果，类似 Apple 产品名展示
